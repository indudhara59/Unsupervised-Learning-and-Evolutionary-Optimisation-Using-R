# Volleyball Case Study

## Directory structure

- `data/` — professor-provided CSV datasets.
- `provided/` — assignment PDFs and the original MALEO template package.
- `scripts/` — the three meaningfully commented R analysis scripts.
- `outputs/` — reproducible figures, completed datasets, and result tables generated by the scripts.
- `reports/source/` — LaTeX report sources and required MALEO class assets.
- `reports/final/` — the three final submission PDFs.

## Reproduce the analysis

Run from the repository root:

```powershell
Rscript scripts/exercise1.R
Rscript scripts/exercise2.R
Rscript scripts/exercise3.R
```

## Rebuild a report

Run from `reports/source/`, for example:

```powershell
pdflatex -interaction=nonstopmode -halt-on-error exercise1_report.tex
pdflatex -interaction=nonstopmode -halt-on-error exercise1_report.tex
```

The PDFs produced in `reports/source/` can then replace the corresponding files in `reports/final/`.

Before submission, add the other group members and accurately update the contribution statement in every report.
